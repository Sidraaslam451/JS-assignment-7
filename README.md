# JavaScript Assignment Dashboard

An interactive JavaScript project that demonstrates basic array and loop assignments with a modern user interface.

## Features

- ✅ Empty Multidimensional Array
- ✅ Matrix Representation
- ✅ Counting from 1 to 10
- ✅ Multiplication Table Generator
- ✅ Fruits Array Traversal
- ✅ Smallest Number Finder
- ✅ Multiples of 5 Generator
- 🌙 Dark Mode with Local Storage
- 📋 Copy Result Button
- 🔔 Toast Notifications
- 📊 Progress Tracker
- 🎨 Animated UI & Glassmorphism Design
- 📱 Responsive Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```text
├── index.html
├── style.css
├── script.js

```
